The following versions of IBM® products are currently licensed to be used as a Supporting Program with Merative™ SPM Platform.

## Merative SPM Platform 8  

- IBM WebSphere Application Server 9.0.5

- IBM MQ 9.2 (Long Term Support)

- IBM Db2 Standard Edition 11.5

- IBM MQ Advanced 9.2 (Continuous Delivery)

## Merative SPM Platform 7 (applicable to 7.0.11.0 iFix7 and subsequent 7.0.11.0 iFix releases) 

- IBM WebSphere Application Server 9.0.5

- IBM DB2 Advanced Workgroup Server Edition 11.1

- IBM Db2 Standard Edition 11.5

- IBM MQ 9.1 (Long Term Support)

- IBM MQ Advanced 9.1 (Continuous Delivery)
